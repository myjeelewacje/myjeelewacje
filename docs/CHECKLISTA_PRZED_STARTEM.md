# Checklista przed startem strony

## Treści

- [ ] Uzupełnić prawdziwy telefon.
- [ ] Uzupełnić prawdziwy e-mail.
- [ ] Dodać adres lub obszar działania.
- [ ] Dodać minimum 6 zdjęć realizacji.
- [ ] Dodać minimum 3 opinie klientów, jeśli je masz.
- [ ] Dopisać dokładne informacje o metodach pracy.

## SEO

- [ ] Ustawić właściwą domenę w `NEXT_PUBLIC_SITE_URL`.
- [ ] Sprawdzić sitemapę `/sitemap.xml`.
- [ ] Sprawdzić robots.txt `/robots.txt`.
- [ ] Dodać Google Search Console.
- [ ] Dodać Google Analytics / Plausible.
- [ ] Założyć Google Business Profile.

## Prawo

- [ ] Dodać politykę prywatności.
- [ ] Dodać zgodę/informację RODO przy formularzu.
- [ ] Dodać dane działalności: firma, NIP, adres, e-mail.

## Techniczne

- [ ] Skonfigurować bazę PostgreSQL.
- [ ] Skonfigurować SMTP.
- [ ] Skonfigurować storage zdjęć.
- [ ] Zrobić test formularza.
- [ ] Zrobić test dodawania realizacji w panelu.
